Sidenav

A minimal kick start for off-canvas menu component. Zero style, just structural. Use the menu above to switch between off-canvas types and then trigger in the bottom menu button.

Minimal style for demo purposes

Usage

You only need a wrapper and inside it add two more elements, a nav and one main content. Then use the modifiers classes to apply the desired transition and that's it, done.

Feel free to change the classes names and tags, but still need keep the properties

If you don't want to use modifiers you also can copy the properties into the modifier and paste to main .sidenav class, just pay attention with some modifiers that need change properties in the content tag.

Javascript

To transition between states you just need to toggle the class .active in .sidenav. If you wanna, I wrote a very basic plugin(?) to do that, feel free to use. You only need to copy the code, and remeber to initialize the plugin using the right reference for your .navbar.

If you change the classes names use $(".newClassName").sidenav()

For the menu icon, you just need to add the class .js-sidenav. Again, feel free to change this on the code.
